# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2024-12-12

### Added
- Initial release of Financial Methodologies Knowledge Base
- **Cash Flow Story** methodology (complete)
  - Philosophy and core concepts
  - Key indicators: Net CF, Marginal CF, Working Capital, Clean CF
  - Power of One framework
  - Use cases and examples (Rebecca's Coffee)
- **Simple Numbers** methodology (structure)
  - Business models: Make, Sell, Do
  - Capital types framework
  - CPR Analysis
- **Theory of Constraints** methodology (structure)
  - Throughput Accounting
  - Five Focusing Steps
  - Constraint identification
- Project structure and documentation
- JSON schemas for validation
- GitHub Actions workflows
- Contributing guidelines

### Infrastructure
- Repository structure
- Automated validation tools
- CI/CD pipeline
- GitHub Pages documentation
