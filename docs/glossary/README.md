# Глоссарий проекта

Глоссарий — единый источник терминов для методологий и моделей.

## Правила
1) Новый термин добавляется только через `data/glossary/<term>.yaml`.
2) Любая методология обязана перечислить используемые термины в front matter:
   glossary_terms: [methodology, model, indicator, ...]
3) Термины в тексте методологий по возможности пишем единообразно.

## Навигация
См. папку `docs/glossary/terms/`.
