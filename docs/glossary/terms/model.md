# Модель (Model)

**Модель — это инструмент принятия решений**, который внутри заданной методологии:
* считает сценарии,
* оценивает последствия изменений,
* сравнивает альтернативы.

### Важно

* модель **подчинена методологии**;
* модель **не существует сама по себе**;
* модель отвечает на вопрос *«что будет, если…»*.

### Пример

**Power of One (+1%)**:
* это модель,
* она не объясняет бизнес,
* она показывает эффект изменения параметров,
* и используется ТОЛЬКО после диагностики.
