# Инструмент моделирования (Modeling Tool)

**Инструмент моделирования — это формализованная модель**, предназначенная для:
* сценарного анализа,
* выбора приоритетов,
* фокусировки управленческих усилий.

### Отличие от модели

* модель — логическая конструкция,
* инструмент моделирования — **реализованная форма** (таблица, калькулятор, сервис).
